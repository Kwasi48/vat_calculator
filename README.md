# vat_calculator

## Description
This is a minimalist VAT calculator localised for Ghana. 

## How it works
This application is used for calculating standard (Exclusive and Inclusive) and Flat Vat calculations. 

## Flat Rate
Uses  a covid - 1% 
VAT - 3%

## Standard Rate
NHIL - 2.5%
GETFUND - 2.5%
COVID - 1%
VAT - 15%
